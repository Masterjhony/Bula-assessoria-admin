**Array de dados (topo do arquivo, antes do componente):**

```
const FOTOS: { src: string; alt: string; caption?: string }[] = []
```

Deixar o array vazio por padrão. O array é a única parte do arquivo que a equipe Bula precisa editar para adicionar fotos. Cada entrada aceita `src` (path/URL da imagem), `alt` (texto acessível obrigatório) e `caption` (legenda opcional exibida no hover e no lightbox).

**Lightbox — estado e lógica:**

- Estado: `const [lightboxIndex, setLightboxIndex] = useState<number | null>(null)`
- Abrir: `setLightboxIndex(idx)` no clique do card
- Fechar: `setLightboxIndex(null)` no botão X e no overlay
- ESC: `useEffect` com `window.addEventListener('keydown', handler)` onde `handler` checa `e.key === 'Escape'`. Cleanup com `removeEventListener` no return do useEffect. Dependência: `[lightboxIndex]`.
- Foco management: `useRef` no botão X do lightbox; quando `lightboxIndex !== null`, chamar `closeButtonRef.current?.focus()` dentro do mesmo useEffect via `setTimeout(..., 0)` para aguardar render.
- `overflow-hidden` no body: adicionar `document.body.style.overflow = 'hidden'` ao abrir e `document.body.style.overflow = ''` ao fechar (cleanup do useEffect).

```
<section id="apartamento-galeria" className="bg-neutral-950 py-14 sm:py-20">
  <div className="mx-auto max-w-7xl px-5 sm:px-8">
    {/* Heading */}
    <div className="mb-10 sm:mb-12">
      <span className="text-[11px] font-bold uppercase tracking-widest text-white/40">
        Prova social · Expertise em campo
      </span>
      <h2 className="mt-2 text-3xl font-black tracking-tight text-white sm:text-4xl">
        O Apartamento foi feito pela Bula
      </h2>
      <p className="mt-2 text-base text-white/55">
        Nossa equipe esteve presente no apartamento do gado. Veja como foi.
      </p>
    </div>

    {/* Estado vazio */}
    {FOTOS.length === 0 && (
      <div className="flex flex-col items-center justify-center rounded-xl border border-white/10 bg-white/5 py-20 text-white/30">
        <Camera className="mb-4 h-12 w-12" aria-hidden />
        <p className="text-base font-medium">Fotos em breve</p>
      </div>
    )}

    {/* Grid */}
    {FOTOS.length > 0 && (
      <div className="grid grid-cols-1 gap-4 sm:grid-cols-2 lg:grid-cols-3 sm:gap-5">
        {FOTOS.map((foto, idx) => (
          <button
            key={idx}
            type="button"
            onClick={() => setLightboxIndex(idx)}
            className="group relative overflow-hidden rounded-xl focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-white/60"
            aria-label={`Abrir foto: ${foto.alt}`}
          >
            {/* Aspect-ratio container */}
            <div className="relative aspect-[4/3] w-full overflow-hidden rounded-xl bg-neutral-800">
              <img
                src={foto.src}
                alt={foto.alt}
                loading="lazy"
                className="h-full w-full object-cover transition-transform duration-200 group-hover:scale-[1.02]"
              />
              {/* Hover overlay com caption */}
              <div className="absolute inset-0 flex items-end bg-black/0 transition-all duration-200 group-hover:bg-black/45">
                {foto.caption && (
                  <p className="translate-y-2 px-4 pb-4 text-sm font-medium text-white opacity-0 transition-all duration-200 group-hover:translate-y-0 group-hover:opacity-100">
                    {foto.caption}
                  </p>
                )}
              </div>
            </div>
          </button>
        ))}
      </div>
    )}
  </div>

  {/* Lightbox */}
  {lightboxIndex !== null && (
    <div
      role="dialog"
      aria-modal="true"
      aria-label="Visualizar foto"
      className="fixed inset-0 z-50 flex items-center justify-center bg-black/90 p-4"
      onClick={() => setLightboxIndex(null)}
    >
      <button
        ref={closeButtonRef}
        type="button"
        onClick={(e) => { e.stopPropagation(); setLightboxIndex(null) }}
        className="absolute right-4 top-4 rounded-full bg-white/10 p-2 text-white transition-colors hover:bg-white/25 focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-white"
        aria-label="Fechar"
      >
        <X className="h-5 w-5" aria-hidden />
      </button>
      <div
        className="relative max-h-[90vh] max-w-[90vw]"
        onClick={(e) => e.stopPropagation()}
      >
        <img
          src={FOTOS[lightboxIndex].src}
          alt={FOTOS[lightboxIndex].alt}
          className="max-h-[80vh] max-w-[85vw] rounded-lg object-contain"
        />
        {FOTOS[lightboxIndex].caption && (
          <p className="mt-3 text-center text-sm text-white/70">
            {FOTOS[lightboxIndex].caption}
          </p>
        )}
      </div>
    </div>
  )}
</section>
```

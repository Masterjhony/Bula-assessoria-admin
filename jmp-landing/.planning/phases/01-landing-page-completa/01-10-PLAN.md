Estratégia de layout:
- Mobile (< sm / < 640px): container flex com overflow-x-auto, scroll-snap-type x mandatory, scrollbar oculta. Cada item é flex-none w-[85vw] scroll-snap-align-start, dando peek ~15% da próxima foto. O container recebe `ref={scrollRef}`.
- sm+ (>= 640px): o container carrossel fica oculto (`sm:hidden`). Um segundo container grid (`hidden sm:grid`) exibe as fotos em 2 colunas (sm) e 3 colunas (lg) — idêntico ao comportamento atual.
- Lightbox: intocado. O clique em qualquer item (carrossel ou grid) chama `setLightboxIndex(idx)`.

Dots (mobile apenas):
- Estado `activeIndex` (número) inicializado em 0.
- `useRef<HTMLDivElement>(null)` chamado `scrollRef` referencia o container do carrossel.
- Dentro de um `useEffect` que depende de `[]`, cria-se um `IntersectionObserver` com `threshold: 0.5` e `root: scrollRef.current`. O observer observa cada filho do container carrossel (querySelectorAll com data-carousel-item). Quando um item entra com intersectionRatio >= 0.5, extrai o index do `data-index` e chama `setActiveIndex`.
- O cleanup do useEffect desconecta o observer.
- Dots são renderizados abaixo do carrossel em `sm:hidden` — um `div` flex justify-center gap-2 mt-4. O clique no dot faz scroll programático: `scrollRef.current.scrollTo({ left: idx * scrollRef.current.offsetWidth * 0.85, behavior: 'smooth' })` — note que cada item tem largura 85vw então o offset por item é `scrollRef.current.offsetWidth * (85/100)` aproximado; use `scrollRef.current.children[idx].scrollIntoView({ behavior: 'smooth', block: 'nearest', inline: 'start' })` para simplificar.

useEffect para o IntersectionObserver (separado do useEffect do lightbox existente):
```
useEffect(() => {
  const container = scrollRef.current
  if (!container) return

  const observer = new IntersectionObserver(
    (entries) => {
      for (const entry of entries) {
        if (entry.isIntersecting && entry.intersectionRatio >= 0.5) {
          const idx = Number((entry.target as HTMLElement).dataset.index)
          if (!Number.isNaN(idx)) setActiveIndex(idx)
        }
      }
    },
    { root: container, threshold: 0.5 }
  )

  const items = container.querySelectorAll('[data-carousel-item]')
  items.forEach((item) => observer.observe(item))

  return () => observer.disconnect()
}, [])
```

O `useEffect` do lightbox existente (com `handleKey` e `document.body.style.overflow`) permanece inalterado.
